## よくある質問 (FAQ)

### Q1. 韓国料理ポケットの基本的な使い方は？

- 料理を「ひらがな」「カタカナ」で検索できます
- お気に入りを追加できます
- 今日のおすすめをランダムで表示できます
- 料理に関するクイズを楽しめます

### Q2. 問題が発生した場合はどうすれば良いですか？

- アプリを再起動してください。
